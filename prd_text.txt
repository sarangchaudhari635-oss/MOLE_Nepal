1. Product Overview
1.1 Problem Statement
Kathmandu Valley generates over 1,200 metric tonnes of solid waste daily, with only a fraction recycled. Industrial waste — metal scrap, plastic offcuts, wood dust, brick residue, carpet scraps — is routinely sent to landfill because factories have no efficient way to find buyers for reusable materials. The informal scrap-dealer network exists but is opaque, geographically limited, and captures excessive margin.
1.2 Solution
MOLE is a digital B2B marketplace where industrial waste generators list their by-products and buyers — recyclers, manufacturers, construction companies — discover and purchase them. An AI image classifier (Gemini Vision) identifies waste material from a photo and auto-suggests matching buyer industries, removing the friction of manual categorisation.
1.3 Core Value Proposition
Waste generators: reduce disposal costs, recover value from scrap
Buyers: source cheap alternative raw materials locally
Nepal: divert industrial waste from Banchare Danda landfill
Informal workers: formalise and monetise existing scrap-collection activity
1.4 MVP Scope (48 hrs)
The hackathon build delivers one complete, live, demo-able user flow: a factory lists waste using an AI-assisted photo scan → the platform matches them with buyers → the buyer sees the listing on a Kathmandu map. Everything else is roadmap.

Leaflet.js map centred on Kathmandu (27.7172° N, 85.3240° E). Custom green pin markers for each listing. Clicking a pin shows a popup with listing summary and 'View Details' link. Uses OpenStreetMap tiles — no API key required.

4.3 Match Engine Logic

5.1 Feature: Waste Image Classifier
This is the single most impressive demo feature. The user uploads a photo of their waste → the Gemini Vision API analyses it and returns structured JSON with material identification, quality assessment, and buyer suggestions. No model training required — this is a prompted API call.
5.2 Prompt Engineering
System Prompt (send with every request)
You are an industrial waste classification expert for Nepal. Analyse the provided image and return ONLY a valid JSON object with no markdown or extra text.
Return this exact structure:
{
  "material_type": "string (e.g. Hardwood offcuts)",
  "category_code": "string (use: wood_biomass | plastic_hdpe | metal_ferrous | metal_nonferrous | textile_wool | concrete_rubble | brick_dust | paper_cardboard | glass_cullet | organic_food | other)",
  "estimated_purity_pct": number (0-100),
  "condition": "Clean | Contaminated | Mixed",
  "price_range_npr": { "min": number, "max": number },
  "buyer_industries": ["array of 3 industry types in Nepal context"],
  "listing_title_suggestion": "string (short, in English and Nepali if possible)"
}
5.3 API Call Implementation
Route: POST /api/ai/classify — accepts multipart/form-data with image field. Converts image to base64, calls Gemini API, parses JSON response, returns to frontend.
5.4 Fallback Strategy
If Gemini API is unavailable or slow during demo, the frontend shows a pre-computed result for a demo image. Store 3 fallback JSON responses in /lib/ai-fallbacks.json keyed by demo image filename. The classify button should NEVER break on stage.

10. Demo Day Script
Rehearse this script verbatim at least 5 times before demo day. Do not freestyle.
Minute 1 — The Problem (speak, no slides needed)
"Kathmandu Valley generates 1,200 tonnes of waste every day. A furniture factory in Bhaktapur throws away 500 kg of wood dust every week and pays NPR 8,000 to dispose of it. Five kilometers away, a biomass energy company would pay for that same wood dust. They have never met. MOLE fixes that."
Minute 2 — Live Demo (practice this on the exact laptop you'll use)
Open MOLE on the live Vercel URL
Show the homepage impact counter — 8 listings, X kg, Y kg CO₂
Click 'Browse Listings' — show Kathmandu map with green pins
Click the Bhaktapur pin — show wood dust listing detail
Click 'List Your Waste' — open form, upload the demo wood dust photo
Click 'Identify with AI' — Gemini returns: Hardwood offcuts, 82% purity, NPR 4–7/kg, 3 buyer suggestions
Show how the form auto-fills from AI result
Log in as demo buyer — show the matched listing appearing in their dashboard
Minute 3 — Impact + Roadmap (memorise these numbers)
"If 50 factories in Kathmandu Valley use MOLE, we estimate 800 tonnes of waste diverted from Banchare Danda landfill per year. That is 400 tonnes of CO₂ equivalent, and NPR 750,000 saved in disposal fees. Post-hackathon we are adding IoT weight sensors for automated listings, eSewa payment integration, and a blockchain certificate so companies can use MOLE data in their ESG reports."
